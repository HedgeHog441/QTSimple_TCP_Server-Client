# QTSimple_TCP_Server-Client

Легкое десктопное приложение на QT для работы с сетевыми протоколами **TCP** и **UDP**. Позволяет гибко настраивать серверную часть и тестировать клиентские соединения в одном интерфейсе.

## 🚀 Возможности

- **Мультисерверность**: Одновременный запуск до 3-х серверов (TCP Threaded, TCP Multiplex, UDP).
- **Гибридный режим**: Возможность выступать в роли сервера и клиента одновременно.
- **Два режима TCP**:
    - *Multi-threaded*: Классическая обработка подключений.
    - *Multiplex (Plex)*: Обработка всех клиентов в одном потоке (повышенная надежность).
- **Поддержка UDP**: Быстрая отправка данных без подтверждения установления соединения.

## 🛠 Настройка и запуск

### Режим Сервера
1. Перейдите на вкладку **Сервер**.
2. Выберите нужный режим:
   - Чекбокс **Plex** — включение мультиплексирования.
   - Чекбокс **UDP** — запуск UDP-сервера.
3. Нажмите **Старт** для запуска или **Стоп** для остановки.

### Режим Клиента
Перейдите на вкладку **Клиент** и используйте консольные команды для управления:


| Команда | Описание | Пример |
| :--- | :--- | :--- |
| `connect [IP:PORT]` | Подключение к TCP серверу | `connect 127.0.0.1:25565` |
| `send [msg]` | Отправка сообщения на TCP сервер | `send Hello World!` |
| `disconnect` | Разрыв текущего TCP соединения | `disconnect` |
| `bind [IP:PORT]` | Установка адреса для UDP посылок | `bind 192.168.0.1:8080` |
| `push [msg]` | Отправка UDP пакета (без подтверждения) | `push Test Data` |
